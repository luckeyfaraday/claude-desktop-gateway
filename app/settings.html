Claude Desktop

Points Claude Desktop's third-party gateway mode at the local gateway. Relaunch Claude Desktop after applying.

"Restore" switches Claude Desktop back to its normal, official Anthropic-hosted mode — use it any time the gateway isn't running and you still want Claude Desktop to work.

Model & routing

The route name must look like a Claude/Anthropic model or Claude Desktop rejects it. Saving while the gateway runs restarts it automatically.
